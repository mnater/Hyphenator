Hinweise zum Gebrauch von Hyphenator.js

Diese Datei gibt Hinweise zum Gebrauch des Skriptes und setzt dieses auch ein und dient so als Testcase. Um Trennungen zu sehen kann die Fensterbreite des Browsers verändert werden.

Version

Diese Datei bezieht sich auf Version 9 des Skripts. Siehe auch die Änderungsliste der jeweiligen Versionen für eventuell neuere Versionen. Diese Version kann auf deutsch-, französisch- und englischsprachigen Webseiten eingesetzt werden.

Verwendung auf einer Webseite

Das Skript wird im <head>-Block einer HTML-Datei eingebunden:
<script src="http://hyphenator.googlecode.com/svn/trunk/Hyphenator.js" type="text/javascript"> </script>
Durch das Einbinden dieses Script wird automatisch das Objekt Hyphenator erstellt.

Anschliessend muss die Methode Hyphenator.run() ausgeführt werden. Das Script wartet automatisch, bis die gesammte Datei geladen ist.
<script type="text/javascript">     Hyphenator.run(); </script>

Das Script sucht automatisch alle Elemente mit der Klasse "hyphenate" (z.B. <p class="hyphenate">) und nimmt die Silbentrennung in diesen Elementen und ihren Kind-Elementen vor. Zur Angabe der Sprache muss das Attribut lang in mindestens einem übergeordneten Element-Tag stehen.

Verwendung als Bookmarklet

Um die Silbentrennung auf «fremden» Seiten durchzuführen (z.B. vor dem Druck der Seite) kann man im Browser das folgende Bookmarklet definieren, welches das Script lädt und die Trennung in allen Elementen durchführt:

javascript: if (document.createElement) { void(head=document.getElementsByTagName('head').item(0)); void(script=document.createElement('script')); void(script.src='http://www.mnn.ch/hyph/v5/Hyphenator.js?bm=true'); void(script.type='text/javascript'); void(head.appendChild(script));}
(diesen Link als Lesezeichen speichern).

Achtung:Je nach Seite kann das einige Sekunden dauern!

Trennung von verschiedensprachigen Dokumenten

Mit dem Attribut lang kann die Sprache eines Textes angegeben werden; so kann sprachenspezifisch getrennt werden. Als Beispiel dienen ein französischer und ein englischer Text von J.J. Rousseau:
FR: Je forme une entreprise qui n'eut jamais d'exemple, et dont l'exécution n'aura point d'imitateur. Je veux montrer à mes semblables un homme dans toute la vérité de la nature; et cet homme, ce sera moi.
EN: I have entered upon a performance which is without example, whose accomplishment will have no imitator. I mean to present my fellow-mortals with a man in all the integrity of nature; and this man shall be myself.
